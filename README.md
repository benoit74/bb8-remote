# bb8-remote
Command your bb8 through node.js from Twitter (via Bluemix) and WebPage

Scan for your BB8 with
$ npm install cylon cylon-ble
$ sudo cylon-ble-scan

Required node modules: (not sure all modules ble, ollie and sphero are required)
- cylon
- cylon-ble
- cylon-ollie
- cylon-sphero
- request
